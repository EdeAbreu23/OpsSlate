# Vision

Homelab Automation Center is a health-aware automation control plane for homelabs.

It is not just a dashboard for scripts. It helps answer:

- Did the job run?
- Did it finish cleanly?
- Did it produce warnings?
- Is it stale?
- Should dependent jobs be allowed to run?
- Where is the log?

The goal is to work alongside existing tools like Unraid User Scripts, Uptime Kuma, ntfy, and Healthchecks instead of replacing them.
