# V1 Scope

## Goals

- Dockerized web dashboard
- Read script status from `status.json` files
- Configure jobs through `jobs.yml`
- Show success, warning, error, and stale states
- Support per-job stale thresholds
- Support warning behavior controls
- Support ntfy notifications
- Provide Unraid-friendly deployment docs

## Non-goals

- Replacing Unraid User Scripts
- Full scheduler in v1
- Editing scripts from UI
- Multi-user authentication
- Unraid plugin packaging
